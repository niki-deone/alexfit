# AlexFit — Personal Trainer Landing Page

A premium multi-page website for a personal trainer business. Built as a portfolio project showcasing modern web design and clean HTML/CSS/JS.

## Live Demo
[View on Vercel →](https://alexfit-omega.vercel.app)

## Pages
- **Home** — Hero, stats, feature highlights, CTA
- **About** — Trainer bio, certifications, values
- **Programs** — Pricing cards, FAQ accordion
- **Reviews** — Rating summary, client testimonials
- **Contact** — Contact form, location, hours

## Features
- Responsive design (mobile, tablet, desktop)
- Smooth scroll animations on reveal
- Working contact form with validation
- Mobile hamburger navigation
- Font Awesome icons
- Google Fonts (Playfair Display + DM Sans)

## Tech Stack
- HTML5
- CSS3 (custom properties, CSS Grid, Flexbox)
- Vanilla JavaScript

## Deploy to Vercel
1. Push to GitHub
2. Import repo on [vercel.com](https://vercel.com)
3. Framework: **Other** (static site)
4. Deploy ✅

No build step required — pure static HTML.

## Project Structure
```
alexfit/
  index.html      — Home page
  about.html      — About page
  programs.html   — Programs & pricing
  reviews.html    — Client reviews
  contact.html    — Contact page
  style.css       — Shared styles
```
